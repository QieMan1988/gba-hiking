# 游戏设计快速参考指南

> 本文档提供游戏设计的关键信息快速查阅，详细内容请参考 `game_design_doc_final.md`

## 核心机制快速参考

### 登山式布局
- 正三角登山式布局，从下往上翻越卡牌
- 每层约2公里，总距离3-26公里
- 累积爬升60-1200米

### 穿越式消除
- 点击：平坦道路（一级地形）
- 长按（1.5s）：上坡路/乱石路/溪流（二级地形）
- 确认框：陡坡/悬崖栈道（三级地形）
- 新手引导期：长按缩短至1.0s，确认框保留

### 连击体系
- 同类型连击：连续消除同类型卡牌（新手期3s，标准期2.5s，挑战期2s）
- 连续攀登连击：连续完成多层攀登（间隔<5s）
- 跨层连击：跨层保持连击（新手期5s，标准期4s，挑战期3s）

### 三阶段体验
- **新手期（0-5小时）**：操作流畅+策略入门（3-7关）
- **中期（5-20小时）**：策略深度+流派探索（7-11关）
- **后期（20-100小时）**：极致追求+终极流派（11-13关）

## 数值系统快速参考

### 五维属性

| 属性 | 范围 | 起始值 | 主要影响 |
|------|------|--------|---------|
| **体能** | 0-100 | 100 | 生命值，为0则游戏结束 |
| **疲劳** | 0-20 | 0 | 行动效率缓冲区，达到20时行动效率降低50% |
| **饥饿** | 0-100 | 100 | 影响体能恢复速率（100%/50%/10%） |
| **口渴** | 0-100 | 100 | 影响体能恢复速率（100%/50%/5%） |
| **心率** | 80-180 | 110 | 影响疲劳积累速率 |

### 体能消耗公式
```
体能消耗 = (徒步距离 / 20) + (累积爬升 / 20)

基准：初始体能 100 = 20 公里徒步 或 2000 米累积爬升
```

### 累积爬升定义
```
累积爬升 = Σ(所有上坡路段的垂直上升高度)
- 上坡路段计入累积爬升
- 下坡路段不计入累积爬升
- 平路（坡度<5%）不计入累积爬升
```

### 疲劳计算公式
```
疲劳积累 = (徒步距离 / 5) + (累积爬升 / 250)

基准：每5公里徒步需要1次休息缓解疲劳
```

### 补给需求公式
```
需水量 = 徒步距离 / 5
需运动饮料量 = 徒步距离 / 10
需巧克力补给次数 = 徒步距离 / 10

基准：每10公里徒步需要2瓶水+1瓶运动饮料+1次巧克力补给
```

## 经济系统快速参考

### 三种货币

| 货币 | 功能 | 获取方式 | 用途 |
|------|------|---------|------|
| **徒步数** | 局外经验 | 徒步距离×2,000 | 局外属性升级 |
| **累积爬升** | 局外稀有货币 | 通关奖励（根据实际爬升） | 兑换高级装备、解锁装备槽位 |
| **环保值** | 局内货币 | 消除环境卡、连击奖励 | 局内商店购买物品+照片卡包 |

### 商店价格

| 商品 | 价格 | 说明 |
|------|------|------|
| 矿泉水 | 25环保值 | 口渴+20 |
| 运动饮料 | 50环保值 | 口渴+15，疲劳-5 |
| 巧克力 | 75环保值 | 体能+15，饥饿+20 |
| 基础照片卡包 | 100环保值 | 必出1张普通照片卡 |

## 关卡设计示例

### 新手教学关（第0关）- 路环东北步行径
- **距离**：4.3公里
- **爬升**：60米
- **层数**：3层（4-3-1张卡）
- **难度**：⭐
- **特点**：最简单的关卡，适合完全新手入门
- **教学目标**：基础穿越机制、体能管理、补给系统、疲劳系统

### 新手关（第1关）- 香港郊野公园入门
- **距离**：10公里
- **爬升**：150米
- **层数**：5层（4-3-3-2-1张卡）
- **难度**：⭐⭐
- **特点**：新手入门关卡，提供连击扶持

### 澳门新手关（第1.5关）- 路环步行径
- **距离**：8.1公里
- **爬升**：80米
- **层数**：4层（4-3-3-1张卡）
- **难度**：⭐⭐
- **特点**：补充关卡，介绍澳门路环岛文化

## 照片卡流派快速参考

### 四大核心流派

| 流派 | 核心效果 | 代表照片卡 | 策略重点 |
|------|---------|-----------|---------|
| **风景连击流派** | 风景卡连击奖励翻倍 | 风景控、山水摄影师 | 优先消除风景卡，建立连击链 |
| **地形连击流派** | 地形障碍卡连击奖励翻倍 | 地形专家、徒步达人 | 规划翻越顺序，利用连击减少消耗 |
| **资源连击流派** | 资源卡连击奖励翻倍 | 资源猎手、自然观察者 | 优先收集资源卡，快速积累环保值 |
| **环境连击流派** | 环境卡连击奖励翻倍 | 环境大师、生态守护者 | 关注环境变化，利用天气优势 |

### 照片卡分层

| 照片卡类型 | 数量 | 获取方式 | 收集时长 |
|-----------|------|---------|---------|
| 普通照片卡 | 30张 | 基础掉落 | 3-10小时 |
| 稀有照片卡 | 15张 | 特定关卡掉落+商店购买 | 10-50小时 |
| 终极照片卡 | 5张 | 第11关通关+挑战模式奖励 | 20-100小时 |

|------|------|---------|

## 游戏流程快速参考

### 单局流程
```
选择路径 → 掷天气骰 → 进入战斗 → 消除卡牌 → 向上攀登 → 
下一层 → ... → 抵达山顶 → 关卡胜利 → 结算成长
```

### 关卡开始流程
```
路线选择 → 掷骰子 → 显示天气 → 开始关卡 → 战斗场景
```

### 关卡胜利流程
```
关卡完成提示 → 资源飞入 → 属性恢复 → 点击进入结算
```

### 关卡失败流程
```
体力耗尽提示 → 属性归零 → 显示累计 → 点击进入结算
```
