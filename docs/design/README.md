## 关键设计要点

### 核心机制

1. **登山式布局**：正三角登山式布局，从底部山脚到顶部山顶
2. **穿越式消除**：通过长按卡牌消除，层消概念，向上攀登
3. **分级翻越**：根据地形难度分级翻越（点击/长按/确认）
4. **连击体系**：同类型连击、连续攀登连击、跨层连击
5. **三阶段体验**：新手期-中期-后期，不同阶段不同体验重点

### 数值系统

1. **五维属性**：体能、疲劳、饥饿、口渴、心率
2. **三种货币**：徒步数、累积爬升、环保值
3. **体能消耗公式**：`体能消耗 = (徒步距离 / 20) + (累积爬升 / 20)`
4. **基准设定**：初始体能 100 = 20 公里徒步 或 2000 米累积爬升

### 关卡系统

1. **真实路线参考**：麦理浩径、澳门路线（路环东北步行径、路环步行径）
2. **累积爬升机制**：上坡计入累积爬升，下坡不计入
3. **补给系统**：每10公里需2水+1运动饮料+1巧克力
4. **疲劳系统**：每5公里需1次休息

### 文化元素

1. **大湾区地理文化**：香港、澳门、深圳、广州、珠海、佛山
2. **自然景观**：大屿山、珠江口、南岭山脉、路环岛
3. **历史文化**：改革开放、香港回归、澳门回归
4. **户外运动文化**：登山文化、骑行文化、徒步文化、露营文化

## 常见问题

### Q: 为什么有两份旧文档？
A: v5.4和v6.0是设计过程中的两个不同版本，v6.0对v5.4进行了数值和关卡系统的更新，但部分内容省略。v7.0最终版完整整合了两份文档的所有内容。

### Q: 应该使用哪份文档作为开发依据？
A: 请使用 `game_design_doc_final.md`（v7.0最终版）作为唯一的开发依据。

### Q: 旧文档还能使用吗？
A: 旧文档已归档，仅供查阅和版本对比，不作为开发依据。

### Q: 如何查看设计变更历史？
A: 请查看 `CHANGELOG.md` 文件，其中记录了所有设计变更的详细信息。

### Q: 为什么路环东北步行径是新手教学关？
A: 路环东北步行径距离短（4.3km）、累积爬升低（60m）、地形平缓，最适合完全新手入门，因此被设定为新手教学第0关。
